วิธีติดตั้ง talking-head-anime-demo บน Windows 10

ฮาร์ดแวร์ที่ต้องมี

  • GPU ยี่ห้อ Nvidia ที่ค่อนข้างใหม่และแรง
    • ผมเคยใช้ GeForce GTX 1080 Ti และ TITAN RTX
    • อัพเดตไดร์เวอร์ให้ทันสมัย
  • เว็บแคม

ซอฟต์แวร์ที่ต้องลงเอง

  • Anaconda Individual Edition
    • https://www.anaconda.com/products/individual
    • ทดสอบโปรแกรมหลังลง
      • เปิด "Anaconda Prompt (Anaconda 3)"
      • เช็คว่าสามารถรันคำสั่ง python และ conda ได้

ดาวน์โหลด talking-head-anime-demo และข้อมูลที่เกี่ยวข้อง

  • เข้า Github ของโปรแกรม
  • ดาวน์โหลดซอร์สโค้ดจากปุ่ม "Clone or download" > "Download ZIP"
  • ขยายไฟล์ใส่ไดเรคทอรีที่ชอบ
    • สมมติว่าใส่ c:\talking-head-anime-demo-master
  • ดาวน์โหลดโมเดลหลักแล้วขยายไฟล์ใส่ c:\talking-head-anime-demo-master\data
  • ดาวน์โหลดโมเดลสำหรับการรู้จำใบหน้าแล้วขยายไฟล์ใส่ c:\talking-head-anime-demo-master\data เช่นกัน

สร้างสภาพแวดล้อมสำหรับใช้ python

  • เปิด Anaconda Prompt (Anaconda 3)
  • เปลี่ยนไดเรคทอรีไปที่ c:\talking-head-anime-demo-master
  • รันคำสั่งต่อไปนี้
    conda env create -f environment.yml
    แล้วรอให้โปรแกรมทำงานจนเสร็จ

ใช้งานจริง

  • เปิด Anaconda Prompt (Anaconda 3)
  • เปลี่ยนไดเรคทอรีไปที่ c:\talking-head-anime-demo-master
  • รันคำสั่งต่อไปนี้
    conda activate talking-head-anime
  • ถ้าจะใช้ manual_poser รันคำสั่งต่อไปนี้
    python app/manual_poser.py
  • ถ้าจะใช้ puppeteer รันคำสั่งต่อไปนี้
    python app/puppeteer.py
