A Quick and Dirty Primer on Exterior Calculus

This is a primer on exterior calculus, which is the study of differential forms and their differentiation and integration. I wrote a similar note before, titled "A Primer on Differential Forms." However, that one is too long and too mathy. This note is more concise and based on materials from Keenan Crane's course on discrete differential geometry; that is, Chapter 4 of the course notes and slides about Exterior Algebra, $k$-forms, Differential Forms, and Exterior Derivatives and Integration. This note is supposed to be more down-to-earth, application-oriented, intuitive, and scraggly than the other one.

1   An Informal Introduction

1.1   Integration in the Lense of Measurements of Infinitesimal Volumes

The main use of exterior calculus is to formulate integration on manifolds. However, let us talk about normal integration that we are familiar with first.

That is, we are familiar with $k$-dimensional integral. We are given a function $f: \Real^k \ra \Real$. And, given a set $U \subseteq \Real^K$, we know how to compute the integral \begin{align*} \int_U f(\ve{u}) \, \dee \ve{u} \end{align*} which is \begin{align*} \int_U f(\ve{u}) \, \dee u^1 \dee u^2 \dotsm \dee u^k. \end{align*}

Notice here that we use Spivak's notation. We index the dimension by superscripts, not subscripts. In other words, we write $\ve{u} = (u^1, u^2, \dotsc, u^k)^T$.

1.2   Integration on Manifolds

We will formally define what a manifold is later. For now, let's just say that they are parameterized shapes: parameterized lines, parameterized curves, and parametermized volumes and so on.

That is, a $k$-manifold $M$ is a set of points in $\Real^n$ that is an image of a set $U \subseteq \Real^k$ under a vector function $\ve{g}: \Real^k \ra \Real^n$ that is bijective and differentiable (i.e., a diffeomorphism).

So, for each $\ve{x} \in M$, there exists $\mcal{u} \in \Real^k$ such that $\ve{g}(\ve{u}) = \ve{x}$.
