Is this actually a flow?

If we have a time-varying vector field $v_t$, we can integrate it to get another time-varying vector field $\phi_t$. The result is quite similar to a flow.

  • $\phi_0(x) = x$
  • $\phi$ is differentiable with respect to $t$.
  • $\phi$ is differentiable with respect to $x$ if $v$ is also differentiable with respect to $x$ (under certain conditions). \begin{align*} \frac{\partial}{\partial x} \phi_t(x) &= \frac{\partial}{\partial x} \bigg\{ x + \int_0^t v_u(\phi_u(x))\, \dee u \bigg\}\\ &= I + \int_0^t \frac{\partial}{\partial x} v_u(\phi_u(x))\, \dee u \end{align*}

However, we are not so sure if it is a flow because we don't know if $\phi_t: \Real^d \ra \Real^d$ is a differeomorphism for all $t$ or not.

Question. Why do we care if it is a flow?

Answer. If it is a flow, we can do a push-forward. \begin{align*} p_t = [\phi_t]_* p \end{align*} A push-forward only works with a flow because we need $\det \phi_t(x) \neq 0$, and this is only so when $\phi_t$ is a diffeomorphism.

Question. So, is $\phi_t$ produced this way a flow?

Answer. Not always. $v$ has to be continuous and cannot change too fast.

Theorem (Picard—Lindelöf). Let $v$ be a time-dependent vector field, and $\phi$ be the time-dependent vector field that is the solution of the initial value problem \begin{align*} \frac{\partial}{\partial} \phi_t(x) &= v_t(\phi_t(x)),\\ \phi_0(x) &= x. \end{align*} If $v$ is continuous in $t$ and Lipschitz continuous in $x$, then, $\phi_t(x)$ is "unique" in the following sense:

for any fixed $t$, if $\phi_t(x) = \phi_t(x')$, then $x = x'$.

Definition. A function $f: \Real^d \ra \Real^d$ is Lipschitz continuous if there is a constant $K \geq 0$ such that $$\|f(x) - f(y)\| \leq K\|x-y\|$$ for all $x,y \in \Real^d$.

So, mathematicians say $\phi_t$ is a flow if $v$ is differentiable in $x$ and the conditions of the Picard—Lindelöf is true.

Question. So, is $\phi_t$ produced this way a flow?

REAL Answer. We are machine learning engineers. Just assume it is a flow.
