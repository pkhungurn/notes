Diffeomorphism

Definition. A function $f: \Real^d \ra \Real^d$ is called a diffeomorphism if it satisfies the following requirements.

  • $f$ is a bijection.
  • $f$ is differentiable, meaning that $\nabla f$ exists and is well-defined.
  • $\nabla f(x)$ is a continous function of $x$.

A bijection is a function that is one-to-one and onto.

If $f$ is a bijection, then it has an inverse function $f^{-1}$.

Theorem (Inverse Function Theorem). Let $f: \Real^d \ra \Real^d$ be a diffeomorphism. Then, $f^{-1}$ is also a diffeomorphism. Moreover, we have that $\nabla f(x)$ is an invertible matrix for all $x \in \Real^d$, and $$ \nabla f^{-1}(y) = \Big( \nabla f\big(f^{-1}(y)\big) \Big)^{-1} $$ for all $y \in \Real^d$.

I don't remember how to prove this, but we should have learned this once in one of our calculus classes.

Corollary. If $f$ is a diffeomorphism, then, for all $x$, $$\det \nabla f(x) \neq 0.$$

Translation

A translation $\texttt{T}_\mu$ is always a diffeomorphism.

  • It is a bijection. It has an inverse $$ \texttt{T}_{\mu}^{-1} = \texttt{T}_{-\mu}.$$
  • It has a well-defined derivative. $$\nabla \texttt{T}_{\mu}(x) = I.$$
  • Its derivative is continuous. (A constant function is continuous as hell.)

Uniform Scaling

A uniform scaling $\texttt{S}_{\sigma}$ is not always a diffeomorphism.

If $\sigma = 0$, then $\texttt{S}_\sigma$ is not a diffeomorphism because $$ \texttt{S}_0(x) = 0$$ for all $x$, so it is clearly not a bijection

If $\sigma \neq 0$, then $\texttt{S}_\sigma$ is a diffeomorphism.

  • It is a bijection. It has an inverse $$ \texttt{S}_{\sigma}^{-1} = \texttt{S}_{1/\sigma}.$$
  • It has a well-defined derivative. $$\nabla \texttt{S}_{\sigma}(x) = \sigma I.$$
  • Its derivative is a constant function, which is continuous.

Scale-Add

Similarly, a scale-add $\texttt{T}_\mu \circ \texttt{S}_\sigma$ a diffeomorphism if and only if $\sigma \neq 0$.
