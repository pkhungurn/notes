Point Transformation Visualization

We can visualize the effect of a 2D vector function ($\Real^2 \ra \Real^2$) by

  1. Color points on the plane so that we get an image.
  2. Apply the function to the image plane.
  3. See the effect of the function on the image.
So, here's an image. Here's a vector function.
$$f(x) = x$$

Here's the effect of the function on the image.

$\xrightarrow{\ \ \ f\ \ \ }$

Each point $x$ is mapped to itself. So, nothing moves!

Definition. The identity function is a function $\mrm{id}: \Real^d \ra \Real^d$ such that $$\mrm{id}(x) = x$$ for all $x \in \Real^d$.
