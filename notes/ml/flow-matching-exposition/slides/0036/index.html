Push-Forward of Gaussian Distributions

What happen if we push-forward isotropic Gaussians with the vector functions we have seen so far?

Translation of Gaussian. $$ [\mrm{id} + \mu_1]_* \mcal{N}(\mu_0, \sigma_0^2 I) = \mcal{N}(\mu_0 + \mu_1, \sigma_0^2 I) $$

$\xrightarrow{\ \ \ \mrm{id} + (1,-1) \ \ \ }$
$\mcal{N}(0,I)$ $\mcal{N}((1,-1),I)$

Uniform scaling of Gaussian. $$ [\sigma_1\, \mrm{id}]_* \mcal{N}(\mu_0, \sigma_0^2 I) = \mcal{N}(\sigma_1 \mu_0, (\sigma_1 \sigma_0)^2 I) $$

$\xrightarrow{\ \ \ 2\, \mrm{id} \ \ \ }$
$\mcal{N}(0,I)$ $\mcal{N}(0,2^2I)$
$\xrightarrow{\ \ \ 0.5\, \mrm{id} \ \ \ }$
$\mcal{N}(0,I)$ $\mcal{N}(0,0.5^2I)$

Scale-add of Gaussian. $$ [\sigma_1\, \mrm{id} + \mu_1]_* \mcal{N}(\mu_0, \sigma_0^2 I) = \mcal{N}(\sigma_1 \mu_0 + \mu_1, (\sigma_1 \sigma_0)^2 I) $$

$\xrightarrow{\ \ \ 0.5\, \mrm{id} + (1,-1) \ \ \ }$
$\mcal{N}(0,I)$ $\mcal{N}((1,-1),0.5^2I)$
