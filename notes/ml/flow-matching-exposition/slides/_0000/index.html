Vector Function

For the current paper we are reading, we are working in a $d$-dimensional space $\Real^d$.

We denote a vector in $\Real^d$ with $x = (x^1, x^2, \dotsc, x^d)$.

This is an unusual convention. However, some really famous textbooks uses it. These textbooks include Spivak's Calculus on Manifolds, which is one of the book I learned multi-variable calculus from.

Definition. A vector function or a vector-valued function is a function $f$ that maps $\Real^n$ to $\Real^m$ for some positive integers $n$ and $m$.

We say that the function $f$ has signature $\Real^n \ra \Real^m$. We also write $f:\Real^n \ra \Real^m$.

Most of the time, we will deal with functions of signature $\Real^d \ra \Real^d$ or $\Real^{d+1} \ra \Real^d$.

Let's talk first about functions of signature $\Real^d \ra \Real^d$. There are two interpretations of this type of functions.

  1. Point transformation.
  2. Vector field.
