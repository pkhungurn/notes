Flow for Gaussian Probability Path

We studied probability path $$p_t = \mcal{N}(\mu_t, \sigma_t^2 I)$$ where $\mu_t: \Real \ra \Real^d$ and $\sigma_t: \Real \ra \Real^+$ are continuous functions.
This path represents a smooth transition from one isotropic Gaussian $\mcal{N}(\mu_0, \sigma_0^2 I)$ to another $\mcal{N}(\mu_1, \sigma_1^2 I)$.

Could this path have been a result of pushing the initial distribution $p_0$ with a flow $\phi_t$?

Well yes.

Let's assume such a flow exists. Then, we would have that $p_t = [\phi_t]_* p_0$. This it means that $$\mcal{N}(\mu_t, \sigma_t^2 I) = [\phi_t]_* \mcal{N}(\mu_0, \sigma_0^2 I).$$ We know that $$\phi_t(x) = \frac{\sigma_t}{\sigma_0} (x - \mu_0) + \mu_t$$ satisfies the above condition. So, actually, the probability path $p_t$ is the result of pushing $p_0$ forward with the above $\phi_t$.

Let's do some sanity checks. Is $\phi_t$ a flow?

  • $\phi_0(x) = \frac{\sigma_0}{\sigma_0} (x - \mu_0) + \mu_0 = x$.
  • Each $\phi_t$ is a scale-add. We know that a scale-add is a diffeomorphism if the scaling coefficient is not zero, and $\sigma_t / \sigma_0 \neq 0$.
  • The derivative of the flow is given by \begin{align*} \nabla \phi_t(x) = \nabla \begin{bmatrix} \sigma_t(x^1 - \mu_0^1)/\sigma_0 + \mu_t^1 \\ \sigma_t(x^2 - \mu_2^1)/\sigma_2 + \mu_2^1 \end{bmatrix} &= \begin{bmatrix} \sigma_t'(x^1 - \mu_0^1)/\sigma_0 + \{\mu_t^1\}' & \sigma_t & 0 \\ \sigma_t'(x^2 - \mu_0^1)/\sigma_0 + \{\mu_t^2\}' & 0 & \sigma_t. \end{bmatrix} \end{align*} Actually, the deritive might not exist if $\sigma_t'$ and $\mu_t'$ do not exist.

So, $\phi_t$ is a flow if and only if $\mu_t$ and $\phi_t$ are differentiable.
