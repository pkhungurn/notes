Probability Path

Definition. A probability path is a family of probability distributions on $\Real^d$ indexed by a real number $t$. Symbolically, we write $\{ p_t: t \in \Real \}$ where each $p_t$ is a probability distribution on $\Real^d$.

Note that a probability path can also be viewed as a function of signature $\Real^{d+1} \ra \Real^+ \cup \{0\}$.

In this point of view, we write $p_t(x)$ as $p(t,x)$.

Back to the idea on a previous page.

What we want is a probability path $p_t$ such that $p_0 = p_{\mrm{noise}}$ and $p_1 = p_{\mrm{data}}$.

Optionally, we want $p$ as a function of signature $\Real^{d+1} \ra \Real^+ \cup \{0\}$ to have nice mathematical properties.

  • $p$ should be continuous, especially in $t$. (We want smooth transition.)
  • $p$ should be differentiable.

These are kind of hard to satisfy if $p_{\data}$ is not continuous in the first place, but we will try.
