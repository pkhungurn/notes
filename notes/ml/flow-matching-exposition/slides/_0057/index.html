Continuity Equation

An important tool to have is a way to check whether a vector field $v_t$ generates a probability path $p_t$ or not.

This tool is called the continuity equation. It comes fluid mechanics.

Let $\rho: \Real^{d+1} \ra \Real$ be time-dependent scalar function and $v: \Real^{d+1} \ra \Real$ be a vector field. The continuity equation is the partial differential equation \begin{align*} \frac{\partial}{\partial t} \rho_t(x) + \mrm{div}( \rho_t(x) v_t(x) ) = 0 \end{align*}

Here, $\mrm{div}$ is the divergence operator. It transforms a vector function $f: \Real^d \ra \Real^d$ to a scalar function of signature $\Real^d \ra \Real$ as follows: \begin{align*} (\mrm{div} f)(x) = \mrm{div}(f(x)) = \frac{\partial}{\partial x^1} f^1(x) + \frac{\partial}{\partial x^2} f^2(x) + \dotsb + \frac{\partial}{\partial x^d} f^d(x) = \sum_{i=1}^d \frac{\partial}{\partial x^i} f^i(x) \end{align*}

In the context of time-dependent vector field $f: \Real^{d+1} \ra \Real^d$, the divergence becomes

\begin{align*} (\mrm{div} f)(t,x) = \mrm{div}(f_t(x)) = \sum_{i=1}^d \nabla_{i+1} f^i(t,x). \end{align*}

So, the continuity equation can be written with my notation as:

\begin{align*} \nabla_1 \rho(t,x) + \sum_{i=1}^d \nabla_{i+1}(\rho \,v^i)(t,x) = 0 \end{align*}

Here's how we can use the continuity equation.

Theorem. Let $v_t$ be a time-dependent vector field. Let $p_t$ be a probabliity distribution. Then, $v_t$ generates $p_t$ if and only if the continuity equation \begin{align*} \frac{\partial}{\partial t} p_t(x) + \mrm{div}( p_t(x) v_t(x) ) = 0 \end{align*} or \begin{align*} \nabla_1 p(t,x) + \sum_{i=1}^d \nabla_{i+1}(p\,v^i)(t,x) = 0 \end{align*} holds.
